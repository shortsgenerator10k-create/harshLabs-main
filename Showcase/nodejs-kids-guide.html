Node.js for Kids! 🚀

You know Python? You can learn Node.js too — it's the same idea!

🤔 What even IS Node.js?

Node.js is like Python — but for JavaScript.

You know how Python lets you write code and run it in a terminal? Node.js does the EXACT same thing — but with JavaScript!

JavaScript used to live only inside web browsers (like Chrome). Node.js broke it free so you can run JS on your computer, just like Python. 🔓

🐍 Python
You write python hello.py
in the terminal to run code
⟷
🟨 Node.js
You write node hello.js
in the terminal to run code
🎮 Fun Fact! When you play a multiplayer game online, or use apps like Discord or Netflix — those are often powered by Node.js running on big computers called servers!
💻 Step 1 — What is the Terminal?
The terminal is a text-based control panel for your computer.
Instead of clicking, you type commands.
Think of it like texting your computer what to do — and it actually does it! 📱
Terminal / Command Prompt
$ node --version ← check if Node is installed
v20.10.0

$ node ← opens the Node.js playground (REPL)
Welcome to Node.js v20.10.0.
Type ".help" for more information.
> now you can type JS code here!
💡 Tip: The > symbol means Node is waiting for you to type. It's like Python's >>> in IDLE — same idea!
👋 Step 2 — Your First Commands
Node.js REPL — try these!
// 👋 Hello World (same as Python's print!)
> console.log("Hello! I am learning Node.js!")
Hello! I am learning Node.js!

// ➕ Math — works just like Python!
> 5 + 3
8
> 10 * 4
40

// 📦 Variables — like Python variables!
> let myName = "Harsh"
> console.log("Hi, my name is", myName)
Hi, my name is Harsh

// 🔁 Ask a question? Use a condition!
> let age = 10
> if (age < 13) { console.log("You're a kid!") }
You're a kid!
🐍 vs 🟨 Python vs Node.js — Same Idea, Different Words
What you want to do 🐍 Python 🟨 Node.js (JS)
Print something print("Hello") console.log("Hello")
Make a variable name = "Alex" let name = "Alex"
If condition if age < 13: if (age < 13) { }
Loop 5 times for i in range(5): for(let i=0; i<5; i++)
Make a function def greet(): function greet() { }
Run a file python hello.py node hello.js
📄 Step 3 — Write a .js File and Run It!
Instead of typing in the REPL every time, you can save code in a file and run it!
Create a file called hello.js — just like Python's hello.py
hello.js — your first Node file!
// hello.js

let name = "Node.js Explorer";
let age = 10;

console.log("Hi! I am a", age, "year old", name);

// Count from 1 to 5 (like a for loop in Python!)
for (let i = 1; i <= 5; i++) {
console.log("Count:", i);
}

// A function (like def in Python!)
function sayHello(personName) {
console.log("Hello,", personName, "🎉");
}

sayHello("Harsh");
sayHello("World");
Terminal — run the file!
$ node hello.js
Hi! I am a 10 year old Node.js Explorer
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5
Hello, Harsh 🎉
Hello, World 🎉
